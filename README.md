# SBD-PR-Caclucaltor
# 💪 Powerlifting PR Calculator

A responsive Powerlifting PR Calculator built using HTML, CSS, and JavaScript.

This web app allows users to calculate their estimated 1 Rep Max (1RM) for:

- 🏋️ Squat  
- 🏋️ Bench Press  
- 🏋️ Deadlift  

It also calculates the overall **SBD Total** (Squat + Bench + Deadlift).

---

## 🚀 Features

- Estimate 1RM using the Epley Formula  
- Predict 1-month progress (5% strength increase assumption)  
- Calculate overall SBD total  
- Fully responsive design  
- Desktop layout:
  - Squat and Bench side by side
  - Deadlift centered below
  - Total centered at bottom
- Mobile friendly vertical layout  

---

## 📊 Formula Used

Epley Formula:

1RM = Weight × (1 + Reps / 30)

Future PR (1 month estimate):

Future PR = 1RM × 1.05

## 📌 How To Use

1. Enter your practice weight.
2. Enter the number of reps.
3. Click the calculate button.
4. View your estimated 1RM and future PR.
5. Calculate your total SBD after completing all three lifts.
